# Python Bitwise Operators 

### Binary AND - &

And operation: Compare the bits, only both is 1, you can get 1, otherwise get 0

**Example**

```
12 & 13 
Result: 12
```

**Explanation**

12: 00001100

13: 00001101

& : 00001100,  it is $12_{10}$
